# 3D Shooter (TypeScript + Babylon.js)

Браузерный шутер от первого лица — играется по ссылке, без установки.
Переписан с однофайлового Three.js на **TypeScript + Babylon.js + Vite**.

> Эта версия живёт в ветке [`babylon-ts`](../../tree/babylon-ts).
> Ветка `main` пока содержит старую vanilla-версию (см. [ROADMAP](ROADMAP.md)).

## Управление

| Клавиша | Действие |
|---|---|
| `WASD` | движение |
| мышь | обзор (нужен pointer-lock — кликни по экрану) |
| `ЛКМ` | стрелять (SMG — автоогонь на удержании) |
| `1` / `2` | пистолет / SMG |
| `R` | перезарядка |
| `E` | открыть/закрыть дверь |
| `Пробел` | прыжок |

## Что внутри

- **FPS-камера**: свой mouse-look, WASD, эллипсоидные коллизии, ручная гравитация и прыжок.
- **Город**: здания со стенами, открываемыми дверьми и крышами, граница, платформа + лестница.
- **Оружие**: пистолет и SMG со своими моделями, магазином, темпом, уроном и отдачей;
  дульная вспышка, hit-marker, всплывающие цифры урона, хедшоты.
- **Тир**: мишени с HP и респавном, счётчик убийств.
- **Собираемые кубики** со счётчиком.
- **Звук**: синтез через WebAudio (без файлов). **Текстуры**: процедурные (DynamicTexture).

## Запуск локально

```bash
npm install
npm run dev      # http://localhost:5173
# прод-сборка:
npm run build && npm run preview
```

## Деплой (GitHub Pages)

Workflow [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml) собирает `dist/`
и публикует в Pages при пуше в `main`. Один раз нужно включить в настройках репозитория:
**Settings → Pages → Source → «GitHub Actions»**.

`vite.config.ts` использует `base: './'` — ассеты грузятся по относительным путям,
поэтому работает и на под-пути проекта (`/3D-shooter-game/`).
